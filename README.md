# Погоны — Тренажёр званий

Простой веб‑тренажёр, где показывается фотография погонов, а пользователь вводит соответствующее воинское звание.

Файловая структура проекта:

- index.html — основной интерфейс
- styles.css — стили
- app.js — логика приложения
- data/ranks.json — список записей (формат ниже)
- assets/ — папка с фотографиями погон (добавьте сюда фото)

Формат `data/ranks.json`:

Массив объектов:

{
  "id": 1,
  "mode": "army" | "navy",
  "rank": "название звания (по-русски)",
  "image": "assets/имя_файла.jpg"
}

Рекомендации по фото:
- Размер: примерно 800×600 или 1200×900; удобен формат JPG или PNG.
- Дайте файлам говорящие имена, например `army_starshiy_leytenant.jpg`.
- Поместите файлы в папку `assets`.

Пример добавления звания:

1) Скопируйте объект-образец в `data/ranks.json` и измените поля.
2) Поместите соответствующее фото в `assets` и укажите путь в поле `image`.

Запуск локально:

Лучше запускать через простой HTTP сервер (фетч локального JSON по file:// не всегда работает).

Python 3:

```bash
python -m http.server 8000
```

Затем откройте в браузере: http://localhost:8000/
